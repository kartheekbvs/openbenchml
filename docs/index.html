Features

Everything You Need to Benchmark ML Models

A complete platform from model upload to public leaderboard ranking.

📦

Multi-Framework Model Upload

Upload models in .pkl, .joblib, .onnx, .pt formats. Supports Scikit-Learn, PyTorch, ONNX, XGBoost, LightGBM, and TensorFlow frameworks out of the box.

📊

Benchmark Datasets

Run evaluations against Iris, Wine, BreastCancer, Digits, CaliforniaHousing, and Diabetes datasets. Built-in train/test splits with proper stratification.

🚀

Comprehensive Metrics

Measure accuracy, precision, recall, F1, MAE, RMSE, R2 score, inference latency, peak memory, CPU usage, and model size — all in one run.

🛡️

Docker Sandbox

User models execute in isolated Docker containers with network isolation, memory limits, CPU quotas, and PID limits. Never run untrusted code directly.

🏆

Public Leaderboards

Three ranking views: by accuracy, by speed (fastest inference), and by size (smallest model). See who builds the best models in the community.

🔑

Full REST API

Programmatic access to all features via JSON API endpoints. JWT authentication with HttpOnly cookies for browser security. OAuth2 compatible.

Get Started

Up and Running in 2 Minutes

🧑‍💻

Development Mode

# Clone the repo git clone https://github.com/kartheekbvs/openbenchml.git cd openbenchml # Install dependencies pip install -r requirements.txt # Run with SQLite (no Docker needed) python run.py # Open in browser http://localhost:8000
🐳

Production Mode

# Clone the repo git clone https://github.com/kartheekbvs/openbenchml.git cd openbenchml # Configure environment cp .env.example .env # Launch all services docker compose up -d # Services running: App: http://localhost:8000 DB: localhost:5432 Redis: localhost:6379 Nginx: http://localhost:80
