How It Works

Your model will be evaluated in an isolated Docker container against the selected dataset. Results include:

  • ✓ Accuracy — Overall prediction correctness
  • ✓ Precision & Recall — Class-level performance
  • ✓ F1 Score — Harmonic mean of precision and recall
  • ✓ Latency — Average inference time per sample
  • ✓ Memory Usage — Peak memory during evaluation
  • ✓ CPU Utilization — Average CPU load during runs

After the benchmark completes, results are automatically posted to the public leaderboard.
